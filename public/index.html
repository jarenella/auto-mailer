How it Works

This project was made when I wanted to make a button on a front-end project that would automatically send and email. After struggling to find an API that could do this that was both completely free and/or not only offered to companies, I decided to make one myself. Is there a free one out there that I couldn't find? Maybe. I didn't actually look that hard. I just realized that this project would be a fun learning opportunity, so I decided to make it!

When this API is called, the following parameters are passed: A users API key, to keep out bots and spammers, the receipient of the email, the return adress email, and the main message body. An email is sent from a 'No-Reply' email adress that sends your message to whoever the chosen receipient is, and informs them that they can get back at the return adress email.

To use this API, a user key is required. Please contact me if you wish to have your own!
